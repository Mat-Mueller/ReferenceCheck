Frequently Asked Questions (FAQ)

1. What is this software, and what does it do?

This software checks citation accuracy in PDFs, ensuring in-text citations match the reference list. It verifies each reference through CrossRef and lets users search Google Scholar for accuracy.

2. How do I use the software?

Upload your PDF, and the software will guide you through citation settings. After the analysis is complete, you’ll see an overview of any mismatched in-text citations and references in your document.

3. What reference style is supported?

The software detects citations and references following the APA style. The results for other reference styles are currently not reliable. Matching may work for other reference styles similar to APA.

4. What is the level of accuracy, and what happens if there’s a mismatch between the in-text citations and the reference list?

While we strive for high accuracy, we cannot guarantee 100% precision, as there is always a chance of mismatches due to limitations in analyzing PDF text and detecting citations and references. If the software detects a mismatch, it will highlight the discrepancy, allowing the user to manually verify and match in-text citations and references.

5. Who is responsible for the software?

This software is a collaborative project between Matthias Müller, Madeleine Buckisch, and Lennart Fischer.
